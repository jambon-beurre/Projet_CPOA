Description

Ce script permet d'interpréter des tables HTML en plus d'un certain nombre d'autres balises. Il reconnaît :

<table border="border">
où border peut valoir 1 ou 0

<tr>
ce qui commence une nouvelle rangée

<td width="width" height="height" bgcolor="bgcolor">
où width et height sont des entiers et bgcolor un code de couleur HTML

Il faut utiliser &nbsp; pour obtenir une cellule vide.
